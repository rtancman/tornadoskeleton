# tornadoskeleton

## How to develop
1. clone repository
2. create python 3.6 virtualenv
3. active virtualenv.
4. setup project
5. config .env
6. run tests
7. run app


```console
git clone tornadoskeleton
python3 -m venv .venv
source .venv/bin/activate
make setup_dev
cp contrib/env-sample .env
make test
make run
```
